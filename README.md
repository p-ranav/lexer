## Quick Start

```bash
$ mkdir build
$ cd build
$ cmake ../.
$ make
$ make check
```

## Highlights

* Lightweight implementation in C++
* Supports UTF-8 multi-byte characters
* 50+ unit tests
* Valgrind reports a clean heap
